# lauyi_back

#### 介绍
layui后台管理系统模板

#### 软件架构
软件架构说明
使用的layui-v2.4.3版本
使用jquery-3.3.1版本 views/lib


#### 安装教程

部署到tomcat下启动

#### 使用说明

1. ipconfig需要修改两个地方 一是layuiadmin下面的config.js 二是views中的script中的menu.js
2. security文件放置的是系统管理的相关文件，框架用的，不允许随意增加与删除
3. service文件放置的是业务逻辑，可随意新增与修改
4. map放置的是Libras（地图）相关文件，如经纬度获取
5. lib是放置第三方组件
6. home是放置欢迎界面
7. images是放置静态资源
8. set是设置当前登录的账户信息
9. template是模板引擎
10.user是登录
11.fonts是字体
12.css是样式文件
